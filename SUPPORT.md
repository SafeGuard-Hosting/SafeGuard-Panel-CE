# Support

How to get help with SafeGuard Panel, depending on what you need.

## 🆓 Community support (free)

- **Bugs & feature requests** → [open an issue](https://github.com/SafeGuard-Hosting/SafeGuard-Panel-CE/issues/new/choose).
  Please use the templates and include your OS, panel version, and logs.
- **Install & how-to questions** → start with the [README](README.md) and the
  built-in knowledgebase inside the panel (every page has contextual help).
- **Security vulnerabilities** → **do not** open a public issue; follow
  [SECURITY.md](SECURITY.md) for private disclosure.

Community support is best-effort by the maintainers and other hosts. There's no
guaranteed response time on free channels.

## 💼 Priority support & managed hosting (paid)

If you run a hosting business and need guaranteed response times, hands-on help,
migrations, or someone to run the infrastructure for you:

- **Priority support & managed hosting** → [safeguardpanel.ca](https://safeguardpanel.ca)
- **Email** → support@safeguardpanel.ca

## 🤝 Licensing the panel (OEM / redistribution)

Running a hosting company and selling reseller hosting is **free**. *Selling or
distributing the panel itself* — bundling it into your own product, reselling it,
or shipping it to your customers — requires a written license first. See
[LICENSE](LICENSE) §11 and email support@safeguardpanel.ca.

## ❓ Which channel?

|---|---|
| Reporting a bug | GitHub issue (bug template) |
| Requesting a feature | GitHub issue (feature template) |
| Reporting a security hole | [SECURITY.md](SECURITY.md) (private) |

| Wanting managed hosting | [safeguardpanel.ca](https://safeguardpanel.ca) |
| Wanting to resell/redistribute the panel | support@safeguardpanel.ca |
